# workflows

AIエージェントによる開発ワークフローの定義。

## ワークフロー一覧

| ワークフロー | 説明 |
|------------|------|
| `pr-gated` | タスクごとにfeatureブランチで開発しPRを作成する |
| `batch-dev` | 全オープンタスクを一括実行し、まとめて1つのPRを作成する |
| `static-site` | Next.jsで静的サイトを構築する（ヒアリング→デザイン選定→エピック単位で構築） |

## プロジェクトへの導入

使用するワークフローの `commands/`, `agents/`, `settings.json` を、プロジェクトの `.claude/` にコピーする。
